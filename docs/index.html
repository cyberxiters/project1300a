About This Project

The Discord Bot Manager is a sophisticated platform that provides advanced authentication, monitoring, and interaction capabilities for Discord servers. This application allows server administrators to manage their Discord bots efficiently.

Key Features

  • Discord Server Management: Connect and interact with your Discord servers
  • Message Templates: Create and manage reusable message templates
  • Campaign Management: Schedule and run messaging campaigns
  • User Targeting: Target specific users or user groups
  • Rate Limiting: Control message frequency to prevent spam
  • Message Logging: Track all sent messages and errors
  • Bot Token Management: Securely store and manage multiple bot tokens
  • Real-time Monitoring: Monitor bot activities in real-time

Technology Stack

  • Frontend: React, TypeScript, Tailwind CSS, shadcn/ui components
  • Backend: Node.js, Express
  • Database: PostgreSQL (with in-memory fallback)
  • Authentication: Passport.js, session-based auth
  • API Integration: Discord.js for Discord API
