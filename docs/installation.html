Local Installation

Follow these steps to install and run the Discord Bot Manager locally:

Prerequisites

  • Node.js (version 18 or higher)
  • npm (comes with Node.js)
  • Git

Step 1: Clone the Repository

git clone https://github.com/cyberxiters/massdmbot.git
cd massdmbot

Step 2: Install Dependencies

npm install

Step 3: Set Environment Variables

Create a .env file in the root directory with the following content:

DATABASE_URL=postgresql://username:password@localhost:5432/discord_bot
SESSION_SECRET=your_secure_random_string

Note: If you don't have a PostgreSQL database, the app will use in-memory storage.

Step 4: Start the Development Server

npm run dev

The application will start on http://localhost:5000

Deploying to Render

Render is a cloud platform that can host your Discord Bot Manager. Here's how to deploy:

Step 1: Create a Render Account

Sign up at render.com using your GitHub account.

Step 2: Create a New Web Service

  • Click "New" and select "Web Service"
  • Connect your GitHub repository
  • Choose the repository with your Discord Bot Manager

Step 3: Configure the Service

Use these settings:

  • Name: discord-bot-manager (or your preferred name)
  • Runtime: Node
  • Build Command: npm ci && npm run build
  • Start Command: npm run start

Step 4: Add Environment Variables

Add these under the "Environment" section:

  • DATABASE_URL: Your PostgreSQL connection string (if using a database)
  • SESSION_SECRET: A secure random string for session encryption
  • NODE_ENV: production

Step 5: Deploy

Click "Create Web Service" and Render will build and deploy your application.

Once deployed, you'll get a URL like https://your-app-name.onrender.com

Setting Up Discord Bot

To use this application, you'll need to create a Discord bot:

Step 1: Create a Discord Application

  • Go to the Discord Developer Portal
  • Click "New Application" and give it a name
  • Go to the "Bot" tab and click "Add Bot"

Step 2: Get Bot Token

  • In the Bot tab, click "Reset Token" and copy your token
  • Add this token in the Discord Bot Manager application settings

Step 3: Set Permissions & Add to Server

  • Go to OAuth2 > URL Generator
  • Select "bot" scope and required permissions
  • Copy the generated URL and open it in your browser
  • Select a server to add your bot to
